# SRGAN Super Resolution

## Status

Local learning-project archive. Public repository pending.

## Source Material

This page summarizes a local deep learning practice package for image super-resolution. The package is based on an
adapted SRGAN PyTorch workflow and includes code, configuration helpers, report material, metrics summaries, and visual
result folders.

The local submission notes explicitly exclude raw datasets, virtual environments, and model weights from the package.

## Technical Focus

- SRGAN training and inference workflow.
- DIV2K-style data preparation.
- Set5 visual comparison outputs.
- PSNR/SSIM metric reporting.
- Baseline and improvement experiment organization.

## Publication Boundary

The public portfolio page can safely describe the workflow and repository structure. Dataset files, trained weights, and
large generated artifacts should stay out of the website unless licensing and storage limits are checked.

## Next Publishing Steps

- Review open-source attribution for the adapted SRGAN baseline.
- Keep only lightweight scripts, configs, README notes, and selected compressed visuals in a future repository.
- Add final metrics only after confirming the exact table and run provenance.
