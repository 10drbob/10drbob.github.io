# Image Captioning with Attention

## Status

Local learning-project archive. Public repository pending.

## Source Material

This page summarizes a local deep learning practice package for image caption generation. The local material includes
dataset processing scripts, an encoder-decoder implementation, attention modules, training code, and batch-evaluation
workflow notes.

Raw datasets and model weights are not copied into this website.

## Technical Focus

- CNN-based image encoder.
- RNN decoder variants.
- Attention-based decoding.
- Greedy decoding and beam-search evaluation.
- BLEU-style evaluation and generated-caption inspection.

## Publication Boundary

Local evaluation files exist, but metric values should be rechecked from a clean run before any public benchmark-style
claim is added. This page intentionally avoids publishing numeric results until that provenance is confirmed.
