# Notes Overview

This is the starting point for 10drbob's Deep Learning, NLP, LLM, and practical AI project notes. The site uses
Markdown and MDX so notes, paper summaries, and project records can grow gradually without a heavy backend.

## Current Sections

- [Deep Learning](./deep-learning/intro): neural networks, optimization, CNNs, RNNs, and Transformers.
- [NLP](./nlp/intro): tokenization, word embeddings, attention, BERT, GPT, and RAG.
- [Paper Reading](./paper-reading/intro): paper summaries, method breakdowns, and reading notes.
- [Projects](./projects/intro): planned and archived project documentation.

## Writing Rule

Each note should answer three practical questions:

1. What problem does this concept or project solve?
2. What are the key methods and implementation details?
3. What did I learn from reading, reproducing, or building it?

Current pages are intentionally lightweight. They will be expanded as the learning and project work becomes more
complete.
