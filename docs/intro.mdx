# Notes Overview

这里是 fdghu 的 Deep Learning / NLP 学习主页。这个网站会用 Markdown / MDX 长期整理课程笔记、论文阅读、项目展示和未来的 Hugging Face Spaces Demo。

## 当前栏目

- [Deep Learning](./deep-learning/intro)：神经网络、优化、CNN、RNN、Transformer 等基础内容。
- [NLP](./nlp/intro)：tokenization、word embedding、attention、BERT、GPT、RAG 等主题。
- [Paper Reading](./paper-reading/intro)：论文阅读摘要、方法拆解和实验记录。
- [Projects](./projects/intro)：深度学习 / NLP 实战项目的展示入口。

## 写作原则

每篇内容都尽量记录清楚三个问题：

1. 这个概念或项目解决什么问题。
2. 它的关键方法和实现细节是什么。
3. 我在复现、实现或阅读中学到了什么。
